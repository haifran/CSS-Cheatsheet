Basic CSS Cheatsheet

CSS Selectors

Selectors Function Description
.class Class Selector Applies to a variety elements
#example ID Selector Single use application for an element
font-family Stylized Text Changes typeface of text on web browsers
rgb() Change colors Three values using red, green, and blue rangin from 0 to 255
hsl() Change colors Using the values hue(color), saturation(intensity), and lightness with ranges from 0 to 360

CSS Specifities

Name Specifity
ID ID selectors have a higher specificity than attribute and Class selectors
Class Class have a higher specificity than attribute selectors but lower than ID selectors
Element Elements are the lower in specificity rank
